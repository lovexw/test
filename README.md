# 文件下载中心

这是一个简单而美观的文件下载中心，提供了一个易于使用的界面来展示和下载各种文件。

## 功能特点

- 清晰的文件列表展示
- 支持多种文件格式（PDF、EPUB等）
- 响应式设计，适配移动端和桌面端
- 简洁美观的用户界面
- 一键下载功能

## 使用说明

1. 将需要分享的文件放入 `book` 目录中
2. 在 `index.html` 中添加对应的文件链接
3. 部署到 Web 服务器即可使用

## 部署方法

1. 克隆或下载本项目到本地
2. 确保所有文件都放在正确的目录结构中
3. 将整个项目目录部署到任何支持静态文件托管的 Web 服务器上

## 目录结构

```
- book/          # 存放各种格式的文件
- index.html     # 主页面
- README.md      # 说明文档
```

## 支持的文件格式

目前支持但不限于以下文件格式：
- PDF 文档
- EPUB 电子书
- 其他文本文档

## 注意事项

- 建议在添加新文件时保持文件名的规范性
- 文件名最好使用中文或英文，避免特殊字符
- 建议控制单个文件的大小，以确保良好的下载体验

## 技术栈

- HTML5
- CSS3
- 响应式设计
